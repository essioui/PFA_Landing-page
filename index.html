API Humain

The API Humain is an innovative interface designed to enhance the online shopping experience.

It allows customers to browse the available product catalog, select items they want, and then send the order to the manager for review.

The manager has the ability to accept or reject orders, as well as add or edit products in the catalog. Thanks to this system, customers can see real-time updates, which improves the user experience and increases the efficiency of order management.

Key Features


Ease of Use

A smooth, user-friendly interface that simplifies browsing and shopping.

Product Catalog Browsing

Customers can view and browse through a catalog of available products, making it easy to find items of interest.

Real-Time Updates

Customers see real-time updates to the catalog, ensuring they have the latest information on product availability.

Order Management

The manager can accept or reject customer orders, providing control over incoming requests.
